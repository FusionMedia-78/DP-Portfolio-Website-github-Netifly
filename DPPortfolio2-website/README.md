# DPPortfolio2-website

Personal portfolio website for David Paull — Creative Director & Visual Storyteller.

## Pages

- **Home** (`index.html`) — Hero, featured projects, stats, client marquee, CTA
- **About** (`about.html`) — Bio, career stats, capabilities, tools, career timeline
- **Projects** (`projects.html`) — Project grid with hover overlays
- **Photography** (`photography.html`) — Masonry gallery with category filters and lightbox
- **Contact** (`contact.html`) — Contact info, form, panoramic image
- **Case Studies** — Individual project pages (Aptus, Pellucid, Village Roadshow, Riper.io, Natural Beauty, ESmart Farms, NFT Campaigns, Doomsday)

## Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no build tools
- Google Fonts: Sora, Outfit, Space Mono
- All images hosted externally via Squarespace CDN
- CSS custom properties for theming
- Intersection Observer API for scroll animations
- CSS columns for masonry photography gallery

## Deployment

This is a static site. Deploy to any static host:

- **Netlify**: Drag and drop the folder, or connect this repo
- **GitHub Pages**: Enable in repo settings → Pages → Deploy from branch
- **Vercel**: Import the repo directly

No build step required.

## Local Development

Open `index.html` in a browser. That's it.
